Музыкальный плеер для Linux с дизайном в стиле Liquid Glass.

Glass Music — это современный, полностью открытый (Open Source) музыкальный плеер, созданный на базе Electron. Наша цель — предложить пользователям Linux изящный и интуитивно понятный интерфейс, вдохновленный дизайном и функциональностью Apple Music с эффектами Liquid Glass (Жидкое стекло).

Ключевые особенности

   Вдохновлен Apple Music: Привычный и красивый дизайн, оптимизированный для настольных систем Linux.
    

   Эффект Liquid Glass: Современные прозрачные и размытые элементы интерфейса.
    

   Полная Открытость (Open Source): Исходный код полностью доступен для просмотра, изучения и модификации.
    

   Конфиденциальность: Glass Music не собирает, не передает и не хранит какие-либо пользовательские данные.
    

   Кросс-платформенность Linux: Сборки для большинства популярных дистрибутивов.

Установка

Glass Music доступен в различных форматах.

1. AppImage (Рекомендуемый) Самый простой способ запустить плеер без установки зависимостей:

    Скачайте последний файл .AppImage со страницы Релизы/Releases.

    Сделайте его исполняемым: chmod +x Glass-Music-*.AppImage

    Запустите: ./Glass-Music-*.AppImage

2. DEB (Debian/Ubuntu/Mint) Для систем на базе Debian:
     sudo dpkg -i  glass-music_1.0.0_amd64.deb 
     (или двойным щелчком по deb-пакету)



3.  Pacman (Arch/Manjaro) Для систем на базе Arch: 


    Cкачайте .pacman и установите через sudo pacman -U glass-music-1.0.0.pacman 
   

4.  Tarball (.tar.gz):

    Для ручного извлечения и запуска: tar -xzvf Glass-Music-*.tar.gz cd Glass-Music ./glass-music

    

Сборка из исходников



Для разработчиков, желающих собрать плеер самостоятельно:

  Клонируйте репозиторий: git clone https://github.com/dvytvs/Glass-Music.git cd Glass-Music

  Установите зависимости (требуется Node.js и npm/yarn): npm install

   Запустите проект в режиме разработки: npm run start

  Создайте свою сборку: npm run dist
